# Guide utilisateur — PSORE V2.4

## Public
L'accueil reste visible, mais les modules détaillés redirigent vers `/login`. La carte publique `/cartographie` affiche des informations limitées.

## Connecté
Les utilisateurs autorisés accèdent à :
```txt
/dashboard
/cartographie/privee
/points-eau
/pluviometrie
/piezometrie
/limnimetrie
```

Les exports CSV/Excel sont disponibles selon le rôle.
