# Guide administrateur — PSORE V2.4

## Connexion
Accéder à `/login`, puis utiliser `ADMIN_EMAIL` et `ADMIN_PASSWORD`.

Changer le mot de passe dans `/mon-compte` après la première connexion.

## Administration
Accéder à `/admin` pour :
- créer des utilisateurs ;
- attribuer des rôles ;
- activer/désactiver des comptes ;
- réinitialiser les mots de passe ;
- consulter les formulaires et journaux de synchronisation.

## Rôles
- Super administrateur : accès total
- Administrateur PTCS : gestion opérationnelle
- DNH/DRHK : consultation, validation, export
- Collecteur : collecte/mise à jour autorisée
- Observateur : consultation limitée
- Public : accès public limité
