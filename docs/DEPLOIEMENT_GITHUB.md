# Déploiement GitHub — PSORE V2.4

## Nouveau dépôt recommandé
```bash
git init
git add .
git commit -m "Initialisation PSORE V2.4"
git branch -M main
git remote add origin https://github.com/<compte>/PSORE_V2_4.git
git push -u origin main
```

## Dépôt existant
Sauvegarder, supprimer les anciens fichiers sauf `.git`, copier cette version, puis :
```bash
git add .
git commit -m "Mise à jour PSORE V2.4 finale"
git push
```
